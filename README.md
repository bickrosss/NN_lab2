# Лабораторная работа №2
Тема: Разработка и исследование нейронных сетей для классификации изображений с применением методов регуляризации **Dropout** и **Batch Normalization**.

## Задачи
 
### 1. Классификация одежды Fashion-MNIST (PyTorch)
Разработали **9 моделей** нейронных сетей с различными архитектурами для классификации 10 типов одежды (футболки, брюки, платья, кроссовки и т.д.) и сравнили их эффективность.
- Использовали деление выборок: **50 000 / 10 000 / 10 000** (train / val / test).
- Сравнили влияние **глубины сети**, регуляризации (**Dropout**, **BatchNormalization**) и выбора оптимизатора (**Adam** vs **Adadelta**).
### 2. Классификация вин `sklearn.load_wine` (Keras)
Обучили модель классифицировать вина по **13 химическим признакам** на 3 класса (всего 178 примеров).
- Провели серию из **6 экспериментов** с разными архитектурами (от однослойной 32-нейронной до комбинации BN + Dropout).
- **Все 6 моделей** преодолели порог точности **94%** на тесте.
### 3. Классификация пассажиров автобуса (CNN, Keras)
Реализовали **свёрточную нейронную сеть** для бинарной классификации пассажиров на «входящих» и «выходящих».
- Архитектура: **3 свёрточных блока** (Conv2D + BatchNormalization + MaxPooling) с ростом фильтров **32 → 64 → 128** + полносвязный классификатор с Dropout(0.5).
- Датасет: **9 081 изображение** 64×64 RGB (загрузка с Yandex Cloud + стратифицированное разделение 80/20).
- **Результат:** точность **96.97%** на проверочной выборке (порог 85%).
- Параметров нейронной сети: **1 142 977** (Total), **1 142 529** (Trainable).
